# setClientChunkGroups()
```tsx
import { setClientChunkGroups } from '@dr.pogodin/react-utils';

function setClientChunkGroups(groups: Record<string, string[]>): void;
```

On the client-side [setClientChunkGroups()] provides to subsequent
[splitComponent()] calls the information on assets generated by Webpack for
each split code chunk; on the server-side it throws an error if called.

In real applications relying on this library it is called automatically
by the [launchClient()] function; however, in Jest tests you may need
to call it manually, if you rely on [splitComponent()] in your render,
without using [launchClient()].

## Arguments

- `groups` &mdash; **Record&lt;string, string[]&gt;** &mdash; the mapping of
  code chunk names to the arrays of paths to Webpack assets generated for
  those chunks.
